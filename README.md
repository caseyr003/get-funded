# Get Funded

This sample project demonstrates how to create a full application with a react frontend and a deployed smart contract. This project was created during the [Ethereum and Solidity: The Complete Developer's Guide course](https://www.udemy.com/course/ethereum-and-solidity-the-complete-developers-guide/)

## Built With

* [NodeJS](https://nodejs.org/)
* [Ethereum](https://www.ethereum.org/)
* [Solidity](https://solidity.readthedocs.io/en/v0.5.8/)
* [React](https://reactjs.org/)

## Prerequisites

* [NodeJS](https://nodejs.org/)
